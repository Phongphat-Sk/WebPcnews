(ข่าวฮาร์ดแวร์) NVIDIA เปิดตัวการ์ดจอ Workstation RTX A4500 20GB และ A2000 12 GB การ์ดจอน้องใหม่สายทำงานตระกูล AMPERE

NVIDIA เปิดตัวการ์ดจอใหม่ในไลน์ Workstation กับ RTX A4500 20GB และ A2000 12 GB สองน้องใหม่ในตระกูล AMPERE สายทำงาน เน้นคำนวณ เน้น Content Creator และงานด้าน AI

NVIDIA RTX A4500 20 GB

A4500 มาพร้อมกับ CUDA Core 7168 คอร์ ใช้โมเดล GA102 GPU มี Tensor Core 224 คอร์ และ RT Core 56 คอร์ มาพร้อมกับความเร็วคล็อก 1.63 GHz มีหน่วยความจำ 20GB GDDR6 มีอินเตอร์เฟซ 320 bit ความเร็วสูงสุด 640Gb/s มี TDP 200W

NVIDIA RTX A2000 12 GB

ในส่วนของรุ่นน้องเล็กสายทำงานอย่าง A2000 12GB นั้นมาพร้อมกับสเปกครึ่งหนึ่งของ A4500 20GB ใช้โมเดล GA106 GPU มี CUDA Core 3328 คอร์ Tensor Core 104 คอร์ และ RT Core 26 คอร์ ความเร็วคล็อก 1.2 GHz หน่วยความจำ 6GB GDDR6 บัส 192-bit ความเร็วสูงสุด 228 GB/s TDP 70W
