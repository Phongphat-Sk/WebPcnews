Manual computer / คู่มือคอมพิวเตอร์

- เปิดเครื่องภาพไม่ขึ้น
- เครื่องดับเอง
- ไม่boot windows
- มองไม่เห็น Hdd เพิ่งซื้อมาใหม่
- เครื่องเปิดไม่ติด?
- เครื่องร้อนง่ายแก้ไงดี?
- ปัญหาต้องตั้งเวลาใหม่
- โปรแกรมค้าง ทำงานไม่ได้ ปิดก็ไม่ได้
- วิธีลงwindows 7
- วิธีลงwindows 8
- วิธีลงwindows 10
- วิธีลงwindows 11
1 CTRL + SHIFT + ESC เปิด TASK MANAGER
2 WINDOWS + D เปิดหน้า DESKTOP
3 WINDOWS + L ล็อคหน้าจอทันที
4 WINDOWS + TAB โชว์หน้าต่างโปรแกรมทั้งหมด และสร้าง NEW DESKTOP
5 WINDOWS + P เลือกจอแสดงผล
6 WINDOWS + SHIFT + S แคปหน้าจอ
7 WINDOWS + PRINTSCREEN แคปหน้าจอแบบรัวๆ
8 SHIFT + DELETE ลบไฟล์แบบถาวร
9 CTRL + SHIFT + N สร้างโฟล์เดอร์ใหม่ / เปิดแท็บไม่ระบุตัวตน
10 รวมปุ่มสำหรับใช้งานเอกสาร
1 ปิดโปรแกรม Startup ที่ไม่จำเป็น
2 ปิดแอพฯ ที่ทำงานอยู่เบื้องหลังระบบ
3 ยกเลิกหรือถอนแอพฯ ที่ไม่ได้ใช้งานบางตัว ด้วยการ Uninstall
4 เรียกคืนพื้นที่ว่างในฮาร์ดดิสก์หรือ SSD
5 ใช้เครื่องมือจัดเรียงไฟล์หรือ Defragment
6 Update Windows ใหม่ล่าสุด
7 อัพเดตไดรเวอร์ของอุปกรณ์ให้ใหม่อยู่เสมอ
8 เปลี่ยน Power Plan
9 ปิดการทำงาน Visual Effect
